# Bivariate Analysis

## Purpose

This module provides tools to calculate the correlation between two sets of data.

## Key Components

*   **`BivariateAnalyisis.cs`**: The main user interface for this module. It gets the number of values and use inputted values
*   **`BivariateAnalysisCalculator.cs`**: Contains the core calculation logic, keeping it separate from the user interaction.
*   **`Correlation.cs`**: An enum that displays the string value of the correlation.

## How to Use

```csharp
// Example of how to do Spearman's Rank
BivariateAnalysis.Start();
```
